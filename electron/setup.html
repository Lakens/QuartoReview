Step 1 — Create a GitHub Personal Access Token

What is a Personal Access Token?

QuartoReview uses GitHub to load and save your Quarto documents. A Personal Access Token (PAT) is a password-like key that lets the app access only the GitHub repositories you choose — without sharing your GitHub password.

  1. Click "Open GitHub token page" below — it opens in your browser.
  2. If asked, log in to GitHub.
  3. The page pre-selects the repo scope. Leave it checked — this gives read and write access to your repositories.
  4. Scroll down and click "Generate token".
  5. Copy the token that appears — it starts with ghp_. You can only see it once.
